# ✅ **CheckList App**

A simple yet elegant **CheckList App** built with **React** and **Tailwind CSS**. It offers features like **dark mode**, **local storage support**, and **task completion tracking** to keep you organized.

🚀 **Live Demo:** [Check it out on Vercel](https://your-vercel-url.vercel.app)

---

## ✨ **Features**
- 📌 **Add & Delete Tasks** – Effortlessly manage your daily to-dos.
- 🎨 **Dark Mode Toggle** – Seamlessly switch between dark and light themes.
- 📂 **Local Storage Support** – Your tasks are preserved even after page refreshes.
- ✅ **Task Completion Tracking** – Mark tasks as done with a simple click.
- ⚡ **Fast & Responsive** – Built with React & Tailwind CSS for a smooth user experience.

---
